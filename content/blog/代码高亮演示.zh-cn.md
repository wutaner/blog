**语法高亮代码块 (Syntax-highlighted Code Block)，指定了语言**

*HTML 示例:*
```html
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>
```

*Python 示例:*
```python
# A simple Python function example
def greet(name="World"):
  """Greets the given name."""
  if not name:
    name = "Anonymous"
  print(f"Hello, {name}!")

# Loop example
for i in range(3):
  greet(f"User {i+1}")
```
